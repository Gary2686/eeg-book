CHAPTER 04

Fundamentals of EEG Signal Processing

EEG 訊號處理基礎：把非線性、非平穩、又被雜訊污染的腦波，拆解到時間-頻率-空間三個維度。

🎯 本章學習目標

  • 了解 EEG 為何屬於「非線性 + 非平穩」訊號，並能說出常用的量化指標（skewness、kurtosis、negentropy、KL 距離）。
  • 掌握五種訊號分段準則 (d₁~d₅) 的精神：自相關、kurtosis、週期圖、AR 預測誤差、AR 頻譜差。
  • 能比較 STFT、Wavelet、Synchro-Squeezed Wavelet、Wigner-Ville 在時頻分析上的優劣。
  • 了解 EMD 的 sifting 過程與 IMF 的概念。
  • 能解釋 Coherency、MVAR、DTF 的關係，並知道 DTF 為何能判斷『誰在帶動誰』。
  • 會使用自適應濾波（LMS、RLS）抑制 EEG 雜訊與偽訊。
  • 掌握 PCA / SVD 的核心想法，能說明它在訊號／雜訊子空間分離的角色。
OVERVIEW

本章地圖

第 4 章是所有後續章節的工具箱。EEG 是非線性、非平穩、低 SNR 的訊號，沒有單一工具可以解；本章把訊號處理界最重要的工具一一搬出來，告訴你它們解的問題是什麼、何時用、怎麼選。

🧭 把工具按目標分組
  1. 特徵化：非線性指標（skewness、kurtosis、entropy）；非平穩 → 短窗 / quasi-stationary。
  2. 分段：5 種 dissimilarity 準則（d₁ 自相關、d₂ kurtosis、d₃ 週期圖、d₄ AR 殘差、d₅ AR 頻譜）。
  3. 時頻分析：STFT / Wavelet / SSWT / Wigner-Ville / EMD。
  4. 連結／因果：Coherency / MVAR / DTF。
  5. 清雜訊：FIR/IIR/Notch；自適應 LMS、RLS。
  6. 子空間：PCA、SVD（後續再加 ICA、CSP）。
4.1

引言 — EEG 訊號處理的工作面

  • 時間域、頻率域、空間域、多向 (multiway) 處理都會用。
  • 把 EEG 視覺化為腦表面圖 (topograph)；分離出單一目標源（如癲癇灶、ERP 來源）。
  • BCI 應用愈來愈仰賴深度學習 + 高頻寬資料 + 雲端協作。
  • 本章符號慣例：t 為連續時間、n 為離散時間樣本（取樣週期 T = 1）。
4.2

媒介的非線性 (Nonlinearity of the Medium)

頭部把皮質下的源訊號非線性地混合到頭皮上。MEG 比較好一點，因為頭組織的磁導率差異小；EEG 則受導電率變化的強烈影響。

常見的非線性指標來自混沌理論：dissimilarity、attractor dimension、最大 Lyapunov 指數 (LLE)。這些可在第 11 章看到實際應用。

4.3

非平穩性 (Nonstationarity)

嚴格平穩要求訊號的整體分布不隨時間改變；通常我們只需要 wide-sense stationary（平均、變異不變）。

EEG 的平均與共變異會隨時間改變（閉眼→睜眼、清醒→入睡、正常→癲癇…），所以我們稱它為「quasi-stationary」，多以 10 秒短窗視為近似平穩。

四個常用的『高階矩 / 熵』指標

指標定義意義
SkewnessE[(x−μ)³] / σ³分布不對稱程度；Gaussian 為 0；右偏為負、左偏為正（依書本定義）。
Kurtosism₄ / m₂² − 3（excess）分布尖度。Gaussian 為 0；尖峰重尾 > 0；扁平 < 0。癲癇 spike 高 kurtosis。
NegentropyH(x_Gauss) − H(x)距離 Gaussian 多遠；ICA 與獨立性的常用指標。永為非負。
KL Distance∫ p₁ log(p₁/p₂) dz兩個分布的差距；非對稱、不滿足三角不等式，但常用作分布相似度。
🧠 熵 (Entropy) 在 EEG 的角色

當大腦同步性提高（例如癲癇接近發作），EEG 的熵會下降；多尺度波動色散熵 (multiscale fluctuation-based dispersion entropy) 是近年新發展的指標。

4.4

訊號分段 (Signal Segmentation)

臨床判讀常把 EEG 切成「同質」段：例如癲癇病人可切成 preictal、ictal、postictal 三段。如何自動找邊界？課文給出五種 dissimilarity 準則：

1
d₁ — 自相關差比較相鄰窗的自相關 r_x(k, m) vs r_x(k, m−1)。
2
d₂ — Kurtosis 差正常節律 kurtosis 低、癲癇/ERP kurtosis 高。
3
d₃ — 週期圖差對自相關做 Fourier 變換得到 S_x(ω,m)，比較相鄰段頻譜差。
4
d₄ — AR 預測殘差用第 m−1 段的 AR 係數預測第 m 段第 1 個樣本；殘差大就是邊界。對殘差再做梯度取極大可得邊界 spike。
5
d₅ — AR 頻譜差以 AR 模型估的頻譜替代 STFT，更穩健的 d₃。

把每個 d(m) 與經驗閾值 η_T 比較，超過就視為邊界。這些方法在無專家標註時尤其有用。

4.5

訊號變換與聯合時頻分析

STFT — 滑動視窗的傅立葉

X(n, ω) = ∫ x(τ)·w(n−τ)·e^(−jωτ) dτ。把 EEG 切成短時間視窗，每個視窗做 FFT，得到一張「時間 × 頻率」的時頻圖。

⚠️ 海森堡測不準
σ²_t · σ²_ω ≥ 1/4：時間越窄，頻率越糊；頻率越精，時間越糊。STFT 用固定大小的視窗，無法同時兼顧高/低頻。

4.5.1 連續 Wavelet Transform (CWT)

Morlet-Grossmann 定義：W(a, b) = a^(−1/2) ∫ f(t)·ψ*((t−b)/a) dt。a 是尺度（控制頻率）、b 是時間位移。

  • 多尺度：高頻用窄視窗、低頻用寬視窗 → 自動適應海森堡。
  • 離散 Wavelet (DWT)：用尺度二倍化 (dyadic) 與多解析度分析 (MRA)；可用快速濾波-降採樣演算法實作。
  • 應用：癲癇偵測（特別是新生兒）、神經元電位建模、ERP/EP 偵測。

🌊 互動：FFT vs Wavelet vs Wigner-Ville 概念比較

🔍 三種時頻分析的「視窗形狀」差別
每個方格代表時頻平面上的一個「分析窗」。比較窗的形狀與大小。

4.5.2 Synchro-Squeezed Wavelet Transform (SSWT)

SSWT 是把 CWT 做後處理：先估計候選瞬時頻率 (IF)，再把能量「擠回」真實 IF 的位置，使時頻圖更清晰。重點公式：

ω(a, b) = −i / W(a, b) · ∂W(a, b)/∂b

由 ω(a, b) 把 W(a, b) 重新分配到最接近的頻率 bin Tf(ωₗ, b)；可重建原訊號。對非平穩、非線性訊號特別有用。

4.5.3 Wigner-Ville 分布 (WVD)

WVD 是 ambiguity function 的二維傅立葉變換，時頻解析度最佳，但對「多分量」訊號會產生交叉項 (cross-terms)。常見處理：

  • Pseudo-WVD (PWVD)：對自相關加窗 w(τ)。
  • Cohen's class：把 WVD 與一個 2D 低通核 Φ(t, ω) 卷積壓抑交叉項。
  • Choi-Williams：採用 Φ(τ, ν) = exp(−ν²τ²/(4π²σ)) 的鐘形核。
4.6

Empirical Mode Decomposition (EMD)

EMD 不假設任何基底函數，直接由資料把訊號分解為一系列「Intrinsic Mode Function (IMF)」。底層理論是 Hilbert-Huang Transform (HHT)。

Sifting 流程

1
找所有局部極值maxima 與 minima。
2
用 cubic spline 連 maxima → 上包絡；連 minima → 下包絡確保整個訊號介於上下包絡之間。
3
計算平均 m₁m₁ = (上包絡 + 下包絡) / 2。
4
取差 d₁ = x(t) − m₁d₁ 理想上應該已對稱、所有 maxima > 0、minima < 0。
5
重複以 d₁ 為新資料 sifting，直到滿足 IMF 條件得到第一個 IMF：C₁ = d₁ₖ。再用 x(t) − C₁ 繼續找下一 IMF。
🌀 IMF 的特性
  • 每個 IMF 表現為「瞬時振幅與瞬時頻率都可變」的簡單振盪。
  • EMD 是自適應、不需先選基底（不像 Wavelet 要選 mother wavelet）。
  • 常見問題：mode mixing → 引入 EEMD (Ensemble EMD) 加白雜訊平均；或 CEEMDAN 提供完整重建與更佳分離。
4.7

連結性、MVAR 與 DTF

Coherency（頻譜同步度）

Coh²ᵢⱼ(ω) = |E[Cᵢⱼ(ω)]|² / (E[Cᵢᵢ]·E[Cⱼⱼ])，其中 Cᵢⱼ = Xᵢ·X*ⱼ 是頻譜互相關。可偵測零時延或固定時延的同步，但無法判別方向。

Granger Causality

「若加上 X 的過去能更好預測 Y，則說 X causes Y」。雙變量 AR 即可計算；多通道時計算成本高且不夠有效。

DTF (Directed Transfer Function)

DTF 由 MVAR 模型的轉移矩陣 H(ω) = L⁻¹(ω)·σ² I 推出：

Θ²ᵢⱼ(ω) = |Hᵢⱼ(ω)|²

若 Θ²ᵢⱼ(ω) > 0，表示電極 i 在頻率 ω 上對 j 有因果影響。DTF 比 Coherency 多了方向資訊，是研究 EEG 內訊息流動的標準工具。

  • 用 LWR 演算法估 MVAR 係數；用 AIC 選階數。
  • 時變版本 SDTF（短窗 DTF）可用於追蹤腦內活動隨時間的傳遞變化，例如手指動作前 1 秒 Cz → C3 的傳遞。
4.8

濾波與去雜訊

EEG 包含 < 100 Hz（多在 30 Hz 以下）資訊，所以高頻雜訊用低通就能去；50/60 Hz 電源用 Notch；放大器頻率響應的非線性可用 equalizing filter 補償。

🛠️ 互動：四種濾波器的頻率響應示意

🎚️ 切換不同濾波器類型，觀察通帶與阻帶
|H(f)| 在頻率域的形狀（橫軸 0–100 Hz）

自適應濾波（Adaptive Filter）— Wiener / LMS / RLS

當雜訊未知，但有參考訊號（如 EOG、ECG、fMRI 觸發訊號）時，可用自適應濾波。

  • Wiener 解：w = R⁻¹p（R 為自相關矩陣、p 為輸入-參考互相關），最小化均方誤差 (MSE)。
  • LMS：w(n+1) = w(n) + 2μ·e(n)·r(n)。條件：0 < μ < 1/λ_max。簡單、運算快，但對相關訊號收斂慢。
  • RLS：用遺忘因子 γ（0 < γ ≤ 1）做指數加權最小平方。收斂快但對實時計算數值較不穩。
⚠️ EEG 常見的偽訊與對策
  • 眨眼 (Eye-blink)：用 FP1、FP2 作參考，自適應消除。
  • ECG：可從心電通道做參考，自適應減除。
  • fMRI 同步紀錄的梯度偽訊：用 MRI 觸發時序做平均模板再扣除。
  • 50/60 Hz：先 Notch；再做接地檢查、阻抗平衡。
4.9

主成分分析 (PCA) 與奇異值分解 (SVD)

DFT、DCT 等次最佳變換的基底與資料無關，無法最大化「分散資訊」。PCA / KLT 則找出一組正交基底，把資料的能量壓到最少的維度。

SVD — 數值線性代數的瑞士刀

任何 N × M 資料矩陣 A 可分解為 A = U Σ Vᴴ。U、V 是正交矩陣，Σ 是奇異值對角矩陣。

  • Σ² = Λ（自相關矩陣的特徵值矩陣）。
  • 奇異值由大到小排列；大奇異值對應主訊號子空間、小奇異值對應雜訊子空間。
  • Moore-Penrose pseudo-inverse：A† = U Σ† Vᴴ，可解最小二乘問題。

PCA = SVD 應用在共變異矩陣

對共變異矩陣做 SVD，等同於 PCA：

  • 每個 eigenvector 是一個 principal component (PC)。
  • 對應的 eigenvalue 反映該方向的變異量 (variance)。
  • 取前 K 個 PC 重建，可降維、去雜訊、白化 (whitening)、做盲源分離的前處理（後續 ICA、CSP 都會用）。
🧮 PCA 在 EEG 的典型用途
  • 濾波：切掉小奇異值對應的子空間，去掉雜訊。
  • 降維：把 64 通道 → 前 10 個 PC，再丟給分類器。
  • 白化：讓各分量變異數相等，是 ICA 演算法（FastICA, Infomax）的前置步驟。
  • BSS：當原始來源彼此不相關但不一定獨立時，PCA 能做盲源分離。
4.10

本章摘要

  • EEG 是非線性、非平穩訊號；要用短窗 + 高階統計 + 自適應方法處理。
  • 分段五準則 d₁~d₅ 可自動找邊界（自相關、kurtosis、週期圖、AR 殘差、AR 頻譜）。
  • 時頻分析：STFT 受 Heisenberg 限制；Wavelet 多尺度；SSWT 後處理；Wigner-Ville 解析最佳但有交叉項；EMD 完全資料驅動。
  • 連結與因果：Coherency 找同步、Granger/DTF 找方向；DTF 由 MVAR 推出，常配 LWR + AIC。
  • 濾波：Lowpass、Highpass、Notch + 自適應 LMS/RLS。
  • PCA/SVD 是子空間分離的瑞士刀，後續 ICA、CSP 都靠它打底。
