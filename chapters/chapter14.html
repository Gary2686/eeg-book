CHAPTER 14

EEG-Based Emotion Recognition and Classification

情緒辨識：從 Valence-Arousal 平面到 DEAP/SEED 資料庫的 AI 模型

🎯 本章學習目標

  • 了解情緒的兩大分類框架：離散模型 (Ekman 6 種) 與維度模型 (Valence–Arousal)。
  • 知道情緒與腦區的關聯：邊緣系統、杏仁體、前額葉皮質 (PFC)、左右不對稱性。
  • 掌握情緒影響 ERP 元件 (EPN、LPP、P300、N170 等) 的方式。
  • 了解 EEG 情緒辨識常用特徵：PSD、頻段功率、半球不對稱性 (asymmetry)、連結性。
  • 認識 DEAP、SEED 等知名情緒 EEG 資料庫。
  • 理解跨受試者 (cross-subject) 泛化的挑戰與常用解法（domain adaptation、transfer learning）。
  • 知道情緒辨識在情感運算 (Affective Computing)、VR、疼痛評估、HCI 上的應用。
OVERVIEW

本章地圖

「情緒」是大腦最迷人的功能之一 — 它影響感知、學習、決策、社交，是 BCI 與 affective computing 領域的核心研究議題。然而，光是「列得出來的情緒」就有 55 種以上，腦區彼此重疊、跨受試者差異大；本章帶你看用 EEG 與 ML 解這道題的標準方法。

🧭 兩個必懂框架
  1. 離散模型 (Discrete)：Ekman 提出的 6 種基本情緒 — anger、disgust、fear、joy、sadness、surprise。
  2. 維度模型 (Dimensional)：Russell 的 Circumplex Model — Valence (正負) × Arousal (激動程度) 的 2D 平面。

情緒的神經生物學基礎

  • 邊緣系統 (limbic system)：海馬、杏仁體 (amygdala)、下視丘、扣帶迴 — 情緒、動機、記憶的中樞。
  • 杏仁體：位於顳葉深部，是恐懼、憤怒、性行為情緒的關鍵。
  • 前額葉皮質 (PFC)：與情緒評估、控制相關；多項研究支持左 PFC 處理正情緒、右 PFC 處理負情緒。
  • 神經傳遞物：多巴胺、正腎上腺素、血清素調節整體情緒『增益』。

多種理論的拉扯

  • James-Lange theory：情緒先是身體變化（心跳加快、口乾）再被『感受』。
  • Direction Model：anger 雖為負，卻是『接近型情緒』，活化左 PFC。
  • Component Process Model：情緒是多個身體與認知元件同步出現的事件。
  • Affective Event Theory：情緒由事件引起，再影響態度與行為。
14.2

情緒對腦的影響 — ERP 與節律變化

關鍵 ERP 元件（按時間順序）

  • P100：~100 ms；恐懼臉 > 開心 / 中性。
  • N100：~100 ms 負波；對情緒臉敏感。
  • N170：~170 ms；對『臉』本身敏感；快樂與恐懼臉振幅更大。
  • VPP (vertex positive peak)：與 N170 同時段、前中央電極為正。
  • EPN (Early Posterior Negativity)：210–350 ms；情緒激發圖比中性大；右枕顳葉明顯。
  • P300：~300–500 ms；可拆 P3a / P3b；對情緒臉的反應有差異。
  • LPP (Late Positive Potential)：400–600 ms 後；中央頂葉；越激動越大；也稱 LPC。
  • Slow Wave (SW)：LPP 之後、持續性的正向慢波，反映持續關注情緒刺激。

節律變化（Zheng et al. 2018 觀察）

  • 正情緒：側顳葉 β 與 γ 上升。
  • 中性：頂枕 α 上升。
  • 負情緒：頂枕 δ 與額葉 γ 顯著上升。

🧠 半球不對稱性 (Hemispheric Asymmetry)

最有名的 EEG 情緒指標之一是 frontal alpha asymmetry：

FAA = ln(αF4 right) − ln(αF3 left)

  • α 越大代表該腦區越「不活躍」（α 阻斷反向解讀）。
  • FAA 越正 → 左前額活動越強 → 多與正向接近情緒關聯。
  • FAA 越負 → 右前額活動越強 → 多與負向迴避情緒關聯。

連結性 (Connectivity)

除了單通道頻譜，左右半球的同步性在情緒切換時也會明顯改變；常用度量：mutual information、coherence、PLI、PLV。

14.3

EEG 情緒辨識的訊號處理與機器學習

典型流程

  1. 誘發：受試者看 IAPS 圖、聽音樂、看影片（DEAP、SEED 採用）。
  2. 預處理：bandpass 1–45 Hz、去除眨眼/肌電 (ICA)。
  3. 特徵：PSD（θ/α/β/γ 頻段）、frontal asymmetry、DE（differential entropy）、近似熵、wavelet 係數。
  4. 特徵選擇：FSOR（feature selection with orthogonal regression）、PCA、互資訊。
  5. 分類器：SVM、Random Forest、CNN、LSTM、GNN。

知名資料庫

DEAP (2012)

32 受試者、40 段音樂 MV、32 通道 EEG + 周邊生理
標註 valence/arousal/dominance/liking
情緒誘發強度有限

SEED (2015, 上海交大)

62 通道 + 中性/正/負 三類影片
同受試者多日重測，可研究穩定性
後續推出 SEED-IV / SEED-V 含更多類別

其他

MAHNOB-HCI、AMIGOS、DREAMER…
多模態（含臉部、ECG、GSR）
標註方式不一

跨受試者挑戰

每個人的 EEG 特徵分佈不同 → 模型在新受試者上表現大幅退化。常見對策：

  • Domain Adaptation：把『新受試者的 EEG』映射到模型熟悉的分佈。
  • Transfer Learning：用一兩段校準資料微調。
  • Adversarial Training：訓練『無法判斷是誰』的特徵表示。
  • Capsule Networks / Graph NN：對腦地形圖更貼近的網路結構。
14.4

其他可量測的生理訊號

EEG 不是唯一的情緒指標；以下都會隨情緒改變：

  • 心跳率 / HRV（壓力、恐懼時上升）
  • 呼吸（快而淺 / 緩而深）
  • 血壓（憤怒、恐懼時上升）
  • 皮膚電阻 (GSR)（汗腺活動，反映 sympathetic 興奮）
  • 體溫、瞳孔大小、肌肉電活動 (EMG，臉、頸、肩)

多模態融合可以提升辨識穩健度，但是 EEG 仍是唯一能直接看到腦活動定位、連結與同步的工具。

14.5

應用面

  • Human–Computer Interaction (HCI)：讓電腦察覺使用者情緒，主動調整互動方式。
  • 遊戲、娛樂：動態調整難度、敘事走向。
  • 學習平台：偵測學習者『卡住』『沮喪』，給予提示或休息。
  • 臨床評估：自閉症 (ASD)、ADHD、Parkinson's 病人臉部情緒辨識能力評估。
  • 記憶研究：情緒刺激對 subsequent memory effect 的影響。
📌 為何 ASD/ADHD 兒童需要？
自閉症兒童可能臉上看似冷靜，心率卻已飆到 120 bpm — 老師若誤以為是『無故的攻擊』，會嚴重影響教學。情緒/生理偵測器能幫助大人理解孩子的真實狀態。
14.6

EEG 評估疼痛 (Pain Assessment)

疼痛是「感覺 + 情緒 + 動機」交織的體驗，與情緒共用大量腦區（前扣帶迴、insula、PFC、thalamus）。書中把疼痛分三類，並列出 EEG 特徵：

  • Phasic（短暫尖銳）：在 sensorimotor 區出現 < 10 Hz 增加（150–400 ms）+ α/β 短暫壓抑（300–1000 ms）+ γ 振盪（150–350 ms）。
  • Tonic（持續數分鐘）：α / β 壓抑；γ 上升不在 sensorimotor 而在 medial PFC。
  • Chronic（慢性）：θ 振盪顯著上升；額區 β/γ 異常。

同步記錄 EEG–fMRI 已被用於研究鐮狀紅血球症 (SCD) 病人疼痛時 DMN / pain processing region 的活動。

14.7

VR 誘發情緒

傳統用照片或聲音誘發情緒效果有限；VR HMD（頭戴顯示器）提供強烈沉浸感，能：

  • 誘發放鬆 / 焦慮 / 恐懼等狀態，做控制實驗。
  • 研究文化、技術背景如何影響情緒反應。
  • 同步紀錄 EEG，能避免外界干擾、創造可重現實驗條件。

近年 IVE（沉浸式虛擬環境）+ EEG 已成為 affective computing 的熱門平台。

14.8

本章摘要

  • 兩大情緒分類框架：離散 (Ekman 6) 與維度 (Valence–Arousal)。
  • 情緒神經機制涉及邊緣系統、杏仁體、PFC；左 PFC 偏正情緒、右 PFC 偏負情緒（valence model）。
  • EEG 上的關鍵 ERP：P100、N100、N170、VPP、EPN、P300、LPP、SW；LPP 對 emotional intensity 最敏感。
  • 常用特徵：PSD、Frontal Alpha Asymmetry (FAA)、connectivity、entropy；常用資料庫：DEAP、SEED。
  • 跨受試者泛化是核心挑戰；transfer learning / domain adaptation / GNN 是主流解。
  • 應用涵蓋 HCI、遊戲、學習平台、臨床評估、疼痛量化、VR 沉浸式情緒誘發。
