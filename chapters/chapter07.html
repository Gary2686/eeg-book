CHAPTER 07

Machine Learning for EEG Analysis

從 k-means 到深度神經網路 — 為腦電訊號量身打造的機器學習工具箱

🎯 本章學習目標

  • 清楚分辨『監督式 / 非監督式 / 半監督式 / 強化學習』的差別與情境。
  • 掌握聚類家族：k-means、ISODATA、DBSCAN、Affinity、Deep Clustering、Fuzzy c-means。
  • 能用直觀比喻說明 SVM 的 margin、support vectors、kernel trick、軟邊際 (C) 的作用。
  • 了解類神經網路從 perceptron、MLP、RBF 到深度網路 (CNN, RNN/LSTM, Autoencoder, VAE, GAN) 的演進。
  • 掌握 Common Spatial Pattern (CSP) 在二類 BCI 的核心公式與用法。
  • 能舉例 HMM、Naïve Bayes、GP、隨機森林等在 EEG 上的應用。
  • 知道 cross-validation、特徵選擇、降維、轉移學習等實務技巧。
OVERVIEW

本章地圖

本章把『把腦電訊號交給機器』的所有工具一次看完。流程通常是：

🛠️ EEG ML Pipeline
原始 EEG → 前處理 (filter / BSS) → 特徵抽取 (PCA, CSP, Wavelet, AR, 熵, Lyapunov…) → 降維 → 分類器 / 聚類器 → 交叉驗證 → 部署。
  • EEG 採樣率低 (100–1000 Hz) → 線上學習可行
  • 多通道 → 時間 / 頻率 / 空間特徵可同時用
  • 資料常 imbalanced、有 outliers → 需健壯分類器
  • 樣本不足時 → 轉移學習、surrogate 資料、VAE / GAN 補強
7.1

引言 — 為什麼 EEG 需要機器學習？

  • 動態環境：BCI 與復健系統要即時適應使用者狀態。
  • 大量資料：累積的健康/疾病 EEG 越多，越能訓練偵測模型。
  • 形態差異明顯：睡眠紡錘、k-complex、癲癇尖波等，分類器擅長辨認。
  • 新興應用：CPS、M2M、IoT、emotive computing 都要嵌入式 ML。
  • 偽影偵測：眼動、心電、肌電的形態與正常腦波不同 → 可由分類器自動標記。
📐 兩大降維工具
  • PCA：X = UΣV^T。允許混合正負，常用於白化、去冗。
  • NMF：X⁺ = W⁺H⁺。所有元素非負；對影像、功率譜等天生非負資料友善。
  • 加上 sparsity 限制 → sPCA、sNMF，特徵更可解釋。
7.2

聚類 (Clustering) — 沒標籤也能找結構

k-means

簡單快速、教科書經典
需指定 k、對初始化敏感、只能球形分群

ISODATA / Gap Statistics

能動態決定 k（分裂 / 合併）
Gap statistics 用參考分佈估最佳 k

DBSCAN

不需 k、可抓任意形狀群、自動標雜訊
對 epsilon / minPts 敏感、難處理密度懸殊

Affinity / Spectral

用特徵向量做譜聚類
能處理非線性決策邊界

Deep Clustering

用 CNN 學特徵 + k-means 互相迭代
無監督下也能學到有判別力的特徵

Fuzzy c-means

每個樣本以隸屬度同時屬於多群
m 控制模糊度，m→1 近似 k-means

k-means 演算法步驟

1
初始化選 C，隨機指定 C 個質心。
2
分配每個樣本歸到最近質心。
3
更新質心各群幾何均值即新質心。
4
檢查收斂within-cluster sum of squares 不再變化即停。
7.3.1~7.3.3

經典分類器：DT、隨機森林、LDA

Decision Tree (DT)

用「最佳屬性切分」遞迴建樹。對人類可解讀、實作簡單，但易 overfit 且只能處理線性可分區段。EEG 應用：運動意念 EEG（上下左右）分類，特徵用 Morlet wavelet 係數、AR 參數、偏度等。

Random Forest

多棵 DT 在隨機特徵子集 + 自助樣本 (bagging) 上訓練，投票決定預測：

  • 抗 overfit、自動排序特徵重要性、對 outlier 不敏感。
  • AdaBoost / Gradient Boosting：序列學習，後者修正前者錯誤。

Linear Discriminant Analysis (LDA)

J_LDA = (w S_b w^T) / (w S_w w^T) ← 最大化『類間散度 / 類內散度』

需要 class 為高斯。常用於 BCI；可加 shrinkage 正規化處理小樣本。書中 [34] 用於健康/失智症 EEG 譜參數分類。

7.3.4

Support Vector Machine (SVM)

SVM 的三大支柱

① Margin 最大化

找一條決策超平面 <w, x> + b = 0，最大化『最近樣本到平面』的距離 2/||w||。

② Support Vectors

真正決定超平面的，只有最靠近邊界的少數樣本（支援向量），其他樣本不重要。

③ Kernel Trick

用 K(x,z) = ⟨φ(x), φ(z)⟩ 隱式映射到高維空間，使原本非線性的資料變線性可分。RBF kernel 最常用。

軟邊際 (Soft Margin)

真實資料常有重疊與雜訊。加入 slack ξ_i：

min ½⟨w,w⟩ + C · Σ ξ_i   s.t. y_i(⟨w,x_i⟩+b) ≥ 1 − ξ_i, ξ_i ≥ 0
  • C 大：嚴格不准誤分 → 接近硬邊際、容易 overfit。
  • C 小：寬鬆容忍 → 對 outlier 健壯，但可能 underfit。
  • C 通常以 cross-validation 選擇。

EEG 應用

  • 眼動偽影偵測（先 ICA 拆，再用 SVM 分類哪個 IC 是眨眼）。
  • 左/右手運動意念 BCI（C3、C4 通道）。
  • EEG 多類分類（wavelet + Lyapunov 特徵）。
  • 62 通道情緒識別（正/中/負；βγ 在側顳葉、δ 在頂枕葉）。
7.3.9

類神經網路 — 從感知器到深度

三段歷史

1958
Perceptron (Rosenblatt)
單層、只能線性可分。Minsky/Papert 指出無法解 XOR，引發第一次 AI 寒冬。
1986
MLP + Backpropagation (Rumelhart, Hinton, Williams)
多層感知器加 backprop，可逼近任意連續函數（普遍逼近定理）。
1988
RBF 網路
徑向基函數作為 hidden activation；常用於迴歸與非線性 BSS。
1998
LeNet (LeCun)
第一個成功的 CNN，用於手寫數字辨識。
2012
AlexNet
在 ImageNet 大勝、引爆深度學習時代。
2014
GAN (Goodfellow et al.)
兩個網路對抗訓練，生成模型新典範。
2015~
ResNet / DenseNet / Transformer
殘差連結、注意力機制，深度可達數百層。

典型 MLP 計算

z_k = f( Σ_j w_kj · f( Σ_i w_ji · x_i + w_j0 ) + w_k0 )

常用 activation：sigmoid（兩端飽和易梯度消失）、tanh、ReLU = max(0, z)（正區不飽和，現代主流）。

常見訓練問題

  • 梯度消失 / 爆炸：sigmoid 兩端飽和 → 用 ReLU、Batch Norm、residual。
  • Overfit：用 dropout、weight decay、data augmentation、early stopping。
  • 樣本不平衡：SMOTE、加權損失、WU 等。

CNN 的四大特性

  • 局部連結 (local connections)：每個神經元只看局部 patch。
  • 權重共享 (shared weights)：同一層內 filter 在影像上滑動。
  • 池化 (pooling)：max-pool 縮小、增加位移不變性。
  • 多層 (深)：低階邊緣→紋理→形狀→物件。

RNN / LSTM

RNN 的隱狀態傳遞時間依賴，但長距離依賴會被梯度淹沒。LSTM 引入 input/forget/output gate 與 cell state，可學長期依賴。應用：語音、EEG 序列、文字。

📺 EEG + DNN 經典應用
  • 癲癇放電 (IED) 偵測（1D CNN，Antoniades 2017）
  • 睡眠分期
  • ADHD、思覺失調、失智症輔助診斷
  • 麻醉深度 / 認知負荷 / 情緒識別
  • 運動意念 BCI（CNN + LSTM 取代手工特徵）
7.3.9.3~4

Autoencoder、VAE、GAN — 生成式工具

Autoencoder (AE)

編碼器 x→h (低維 code)，解碼器 h→x̂，最小化 ||x − x̂||²。應用於：

  • 降維、特徵學習（取代 PCA）
  • Denoising AE：輸入加雜訊、輸出乾淨版 → 學去噪。
  • Sparse AE：限制 hidden 大多為 0，特徵更可解釋。
  • Contractive AE：對輸入小變動更不敏感。

Variational Autoencoder (VAE)

把 latent 視為機率分佈（常 N(μ, Σ)）。損失 = 重建誤差 + KL(q(z|x) || N(0, I))。是生成模型，可採樣產生新樣本。EEG 應用：surrogate data 擴增、樣本不足時補強訓練集。

GAN (Generative Adversarial Network)

Generator G 嘗試造出像真實的樣本；Discriminator D 嘗試分辨真假。
訓練目標是 minimax，理想時 G 完全模仿真實分佈、D 對所有輸入都輸出 0.5。

Recent DNN Architectures

LeNet → AlexNet → ZefNet → VGG → GoogLeNet (Inception) → ResNet → DenseNet → Transformer-based 模型（如 EEG-Conformer）。
Spiking NN (SNN)：用 spike timing 編碼，硬體上更省電，更像生物神經元。ENet、YOLOv4 等則追求即時應用。

7.4

Common Spatial Pattern (CSP) — BCI 招牌特徵

CSP 是兩類 EEG 分類（如左/右手運動意念）最成功的空間濾波方法之一：

w_CSP = arg max (w^T C_1 w) / (w^T C_2 w)

等價於解 C_2^{-1} C_1 的特徵值問題。w 對應的最大／最小特徵值方向就是最能分辨兩類的『空間 pattern』。

使用流程

1
帶通濾波聚焦於 μ/β 節律 (8–30 Hz)。
2
計算 C_1, C_2兩類各自的空間共變異矩陣。
3
解特徵值問題取最大/最小幾個特徵向量作為 CSP filters。
4
投影 + 取 log(var)每個 trial 得到固定維度特徵向量。
5
送入 LDA / SVM 分類完成左右手意念分類。
⚠️ CSP 的弱點與補救
容易 overfit、對 outlier 敏感、假設高斯。常用補救：
• Tikhonov 正規化（||w||² penalty，書中報告為最佳）
• Diagonal Loading、Composite CSP、Invariant CSP
• L1-norm CSP（抗 outlier）
• NCSP（不需高斯假設）；多類則用 JAD 擴展為 multiclass CSP
7.3.14

Hidden Markov Model (HMM)

HMM 是描述『序列』的狀態機。每個狀態都帶有觀察的機率分佈。書中以 ECG 為例：bar→p→QRS→t→bar→b→… 的次序若正確出現，就判斷『健康』。

HMM 三大問題

問題內容演算法
① 評估給定 λ=(A,B,π) 與觀察 O，計算 p(O|λ)Forward / Backward
② 解碼找最可能狀態序列 Q*Viterbi
③ 學習由 O 與狀態數估出 λBaum-Welch (EM)

EEG 應用

  • BCI 中分類四種運動任務（AR 係數 + PCA + HMM）。
  • 線上分類左/右手意念（GMM 描述每個狀態）。
  • 用 ApEn、Kolmogorov complexity 做精神疲勞 HMM。
  • 同步 EEG-fMRI 睡眠時的腦網絡動態（HMM）。

其他重要工具速覽

  • Naïve Bayes：特徵獨立假設、超快、樣本需求低。
  • GMM：以混合高斯近似 likelihood，常與 EM 一起用；新生兒癲癇偵測表現佳。
  • Logistic Regression：估事件發生機率，可解釋性佳。
  • Reinforcement Learning：BCI 線上適應、自動神經回饋。
  • Gaussian Processes (GP)：函數上的機率分佈，可給出預測信心；e-health 個人化醫療。
  • Graph Convolutional Networks (GCN)：把腦連結圖當輸入，直接學圖嵌入。

實務技巧

  • Cross-validation (k-fold)：估計泛化能力的標準做法。
  • 特徵選擇：相關性、互訊息、L1 正規化、樹模型的 feature importance。
  • 處理不平衡：SMOTE 上採樣、cost-sensitive SVM、weighted undersampling。
  • 轉移學習：凍結預訓練網路、微調最後幾層；EEG 資料稀缺時必備。
7.5

本章摘要

  • 機器學習 = 學習 + 聚類 / 分類 / 預測；可分為監督、非監督、半監督、強化、深度與集成學習。
  • EEG 的好夥伴：PCA / ICA / CSP 做特徵，SVM / LDA / 隨機森林 / DNN 做分類。
  • SVM 三支柱：margin、support vectors、kernel；軟邊際靠 C、ξ 調節。
  • 深度學習：CNN 處理空間、RNN/LSTM 處理時間、AE / VAE / GAN 生成模型；ReLU 是現代標配。
  • CSP 在二類 BCI 至今仍非常強悍，但需正規化以對抗 overfit 與雜訊。
  • HMM 擅長序列偵測；Naïve Bayes / GMM 簡單卻有效；GP / GCN 是新興工具。
  • 實務上要重視 cross-validation、特徵選擇、樣本平衡、轉移學習與資料擴增。
  • 未來：分散式 / 合作式學習、即時 DRL、surrogate data、可解釋 AI、情緒/社會智能整合。

EEG 的機器學習旅程，從傳統線性方法走到能即時生成、模擬人腦的深度模型。下一個十年，『可解釋』與『嵌入式即時』會是兩個關鍵字。
