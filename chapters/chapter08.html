CHAPTER 08

Brain Connectivity and Its Applications

腦連結性：從相位同步到圖論，看大腦如何「組隊工作」

🎯 本章學習目標

  • 區分「功能性連結 (functional connectivity)」與「有效性連結 (effective connectivity)」。
  • 用白話說明 Coherence、Phase Locking Value (PLV) 等同步指標的意涵。
  • 理解 Granger Causality / DTF / PDC 為什麼能描述「誰影響誰」。
  • 掌握圖論的核心指標：度 (degree)、集群係數、最短路徑、小世界網絡。
  • 知道什麼是「網絡樞紐 (hub)」，及為什麼樞紐失能會造成嚴重的腦功能異常。
  • 了解腦連結性在 Alzheimer's、MCI 等臨床診斷上的角色。
  • 對體積導體效應 (volume conduction) 等「假同步」的陷阱有警覺。
OVERVIEW

本章地圖

傳統 EEG 看的是單一電極的振幅變化。但近年神經科學的核心觀念是：「腦功能來自腦區之間的合作」。本章把腦看成一個由節點（電極或腦區）+ 邊（連結）組成的網絡，並介紹用 EEG 估計這些連結強度與方向的工具箱。

🧭 三層思考架構
  1. 是否有連結？ → 相關、Coherence、PLV、Phase-Slope Index。
  2. 方向是誰指向誰？ → Granger Causality、DTF、PDC。
  3. 整體網絡長什麼樣？ → 圖論：節點度、集群、路徑長度、樞紐、小世界。
8.1

引言 — 腦不是一堆獨立區域

腦連結性研究的根可以追溯到 Santiago Ramon y Cajal（1852–1934）與 Brodmann（1868–1918，把皮質劃分成 52 區）。Wilder Penfield 用電刺激為這些區域畫上功能標籤。Sporns 把這個概念延伸成腦連結組 (brain connectome)，現代神經科學把腦看成一個彼此通訊的網絡。

連結性研究可以從幾個維度劃分：

結構性 vs 功能性

結構性：解剖上的纖維束（DTI/MRI 可看到）
功能性：訊號上的相關／同步
結構不見得用，功能也可能跨遠處發生

靜態 vs 動態

靜態：不隨時間改變的「連線圖」
動態：隨任務或狀態變化的時變連結
EEG 最擅長看動態

微觀 / 中觀 / 巨觀

微觀：單一神經元
中觀：皮質柱
巨觀：腦區，EEG 主要在這層
⚠️ 重要陷阱：體積導體 (Volume Conduction)

EEG 訊號會像電流穿過導體一樣，同一個源在好幾個電極都被看到，且相位差近乎為零。這會造成「假同步」假象。處理連結性時必須能分辨體積導體 (零相位差) 與真正的功能連結 (有時間延遲)。

→ 因此單純的 Pearson 相關不夠用；要用會看「相位」的指標如 coherence、PLV、PSI。

8.2

Functional vs Effective Connectivity

Functional Connectivity 功能性連結

兩個區域的訊號時間相關就算數
不分方向，只看「有沒有同步」
常用：相關係數、Coherence、PLV
無法告訴你「誰影響誰」

Effective Connectivity 有效性連結

描述方向性的因果影響
A 影響 B，但 B 不一定影響 A
常用：Granger Causality、DTF、PDC
需要時間序列模型 (AR / MVAR)

功能性連結 就像同步跳舞； 有效性連結 則像領舞者帶動跟隨者。

8.3

Coherence 與 Phase Locking Value

📐 Coherence 同調性

Coherence 是「頻率域的相關性」。在頻率 ω 上，兩通道 i, j 的同調性定義為：

Coh²ij(ω) = | E[Cij(ω)] |² / ( E[Cii(ω)] × E[Cjj(ω)] )
其中 Cij(ω) = Xi(ω) · Xj*(ω) 是交叉頻譜 (cross-spectrum)。

白話：把訊號用 FFT 拆成不同頻率成分後，看兩通道在每個頻率上「振幅+相位」的相關有多強。值介於 0–1，越接近 1 越同步。

🔄 Phase Locking Value (PLV)

PLV 只看「相位差是否一致」，忽略振幅。如果每次都差固定 π/4，PLV 接近 1；若亂跑接近 0。

  • 對振幅的雜訊較不敏感。
  • 但仍會被體積導體污染（零相位差也是「鎖相」）。

⏳ Phase-Slope Index (PSI)

為了對抗雜訊與獨立雜訊的影響，提出 PSI：cause 必須早於 effect，所以相位的斜率反映方向性。

📌 直覺記法
Coherence 看「有沒有一起變」；PLV 看「相位有沒有鎖住」；PSI 進一步看「斜率」推方向。
8.4

Granger Causality 與 Directed Transfer Function (DTF)

🧠 Granger Causality 的核心觀念

1969 年 Clive Granger 提出（後來拿諾貝爾獎）：

若用 y(t) 的過去來預測 x(t)，比只用 x(t) 自己的過去更準，就說 y 是 x 的 Granger 原因。

注意它不是「真實的」因果，只是「資訊上能預測」。但仍是 EEG 連結方向最常用的概念。

1
單變量 AR 模型e(t) = x(t) − Σ a(i)·x(t−i)；σ²e 是只用 x 過去預測 x 的誤差。
2
雙變量 AR 模型e₁(t) = x(t) − Σ a₁₁(i)x(t−i) − Σ a₁₂(i)y(t−i)；如果加入 y 之後誤差σ²e1 顯著變小⋯⋯
3
Granger 因果指數GCI = ln(σ²e / σ²e1)。值越大代表 y 越能幫忙預測 x，y → x 越強。

🔁 從 Granger 到 MVAR 與 DTF

多通道時，逐對 Granger 太貴。改用 多變量自迴歸 (MVAR) 一次估全部通道的係數矩陣，再轉成頻率域，得到傳輸矩陣 H(ω)。

DTF (Directed Transfer Function)：

Θ²ij(ω) = | Hij(ω) |² ；正規化後 γ²ij(ω) = | Hij(ω) |² / Σm | Him(ω) |²

γ²ij 介於 0–1，且每列總和為 1。優點：能在兩個區域頻譜重疊時仍區分方向（這是 Coherence 做不到的）。短時 DTF (SDTF) 可以追時變方向，常用於手指運動 BCI。

8.5

Partial Directed Coherence (PDC)

DTF 看的是「所有路徑」(包含間接傳遞)。Baccala & Sameshima (2001) 提出 PDC，只看『直接』影響：

Pij(ω) = Aij(ω) / √( aj*(ω)·aj(ω) )

其中 A(ω) 是 MVAR 係數的傅立葉轉換。直觀來說：

DTF

A → B → C 時，A 對 C 也算有連結
無法區分直接 vs 間接

PDC

A → B → C 時，A 對 C 的 PDC ≈ 0（只有 B 對 C 才有）
較敏感於雜訊與模型階數選擇
💡 實務
實際研究常同時報告 DTF 與 PDC。前者反映整體資訊流，後者反映直接路徑。
8.6

圖論 (Graph Theory) 與小世界網絡

有了「節點」與「邊」的權重後，可以把腦變成一張圖，計算它的拓撲特徵。

指標意義白話比喻
度 (Degree)節點連到幾個鄰居朋友數量
集群係數 (Clustering Coefficient)「我的朋友彼此也是朋友」的比例朋友圈緊密程度
最短路徑長度 (Path Length)兩節點之間平均要跳幾步六度分隔
中心性 (Centrality)節點在資訊流通中的重要性誰是「樞紐」
小世界性 (Small-Worldness)高集群 + 短路徑「熟人」「捷徑」兼具

🌍 小世界網絡 (Small-World)

Watts & Strogatz (1998) 經典結果：

  • 規則格子 (Regular)：高集群、長路徑。
  • 隨機網絡 (Random)：低集群、短路徑。
  • 小世界 (Small-World)：高集群 + 短路徑，兼具二者之長。

大腦正是個典型的小世界網絡。研究指出，Alzheimer's、思覺失調等病患的腦失去小世界性（集群下降或路徑變長），可作為診斷依據。

8.7

網絡樞紐 (Hubs)

圖論中度極高的節點稱為 樞紐 (hub)。在腦網絡中，已知的樞紐包括：

  • 後扣帶皮層 (PCC) / 楔前葉 (precuneus)
  • 內側前額葉皮層 (mPFC)
  • 後頂下回 (inferior parietal)
  • 外側顳葉 (lateral temporal)

它們是預設模式網絡 (Default Mode Network, DMN) 的核心。

⚠️ 樞紐失能假說
Alzheimer's 早期常見 PCC / 楔前葉的代謝下降與連結減少。因為樞紐連結最多，它出事整個網絡都會崩。就像機場大罷工，少一個小航點沒事，但 JFK 一關，全美航運打結。
8.8

臨床應用 — Alzheimer's 與 MCI

連結性分析在神經退化疾病診斷上有大量應用。Huang et al. (2010) 比較了 AD、MCI、正常受試者的腦網絡：

連結類型AD 的變化MCI 的變化
顳葉內 (within-temporal)↓ 顯著下降無顯著差異
額葉內 (within-frontal)↑ 增加（代償假說）輕微增加
頂葉-枕葉↑ 顯著增加（代償）↑ 但較弱
額葉-枕葉無顯著差異↓ 顯著下降
左右半球之間顯著破壞較不明顯
📖 代償 vs 破壞

AD 的腦連結變化不是「全部下降」，而是「某些下降、某些反而上升」。後者解讀為大腦試圖代償，調用更多認知資源。這也是為什麼用單一通道很難診斷 AD，需要看整張網絡。

🛠 其他應用

  • 癲癇 (Seizure)：用 ECoG/EEG 找癲癇灶與傳播路徑。
  • BCI：SDTF 用於分類左右手運動／想像。
  • Parkinson's：追蹤震顫 (tremor) 與運動皮層的相位耦合。
  • Hyperscanning：同步記錄多人 EEG，研究合作／競爭情境的「跨腦」同步。
  • 謊言偵測：結合 PDC 與 PPG，分析受訪者是否說謊。
8.9

本章摘要

  • 腦連結性分為結構 vs 功能、靜態 vs 動態、功能性 vs 有效性。EEG 最擅長動態的功能性與有效性分析。
  • 分辨真同步與體積導體假同步是首要任務 — Coherence / PLV / PSI 比 Pearson 相關更可靠。
  • Granger Causality 用「過去能否預測」定義方向；MVAR + DTF 適用多通道；PDC 進一步區分直接 / 間接。
  • 圖論四大指標：度、集群係數、最短路徑、小世界性。腦是典型小世界網絡。
  • 網絡樞紐失能是許多神經疾病（AD、思覺失調）的共同特徵。
  • 應用範圍橫跨 AD/MCI 診斷、癲癇定位、BCI、Hyperscanning。
