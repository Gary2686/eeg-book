CHAPTER 06

Chaos and Dynamical Analysis

當訊號像雜訊卻又不是雜訊 — 用混沌與動力學工具捕捉腦的『深層規律』

🎯 本章學習目標

  • 理解『混沌』不是純隨機 — 是『確定性規則卻看似隨機』，與雜訊有本質差異。
  • 掌握熵 (Entropy)、Kolmogorov 熵、多尺度離散熵 (MFDE)、近似熵 (ApEn) 的意義與差別。
  • 能說明 Lyapunov 指數的物理意義與如何從時間序列估出。
  • 掌握吸引子的概念，能畫出相空間，理解時間延遲 τ 與嵌入維度 m 的選擇。
  • 了解碎形維度 D_f、相關維度 D_r、Lyapunov 維度 D_L 之間的關係。
  • 能舉例說明 DFA、RQA、Bispectrum 在 EEG 的應用情境。
OVERVIEW

本章地圖

EEG 訊號表面像雜訊，但其下藏著大量結構。混沌 (chaos) 與動力學分析提供一組工具，可在沒有明確系統模型的情況下，從訊號本身估出系統的『複雜度指紋』。

🧭 核心觀念地圖
  • 熵類：Entropy → Kolmogorov → Multiscale Fluctuation Dispersion Entropy → Approximate Entropy
  • 幾何類：相空間吸引子 → Fractal / Correlation / Lyapunov dimension
  • 動力類：Lyapunov exponent（看軌跡是否發散）
  • 跨尺度類：DFA（去趨勢，估冪律 scaling exponent）
  • 圖形類：Recurrence Plot / RQA（系統何時回到過去狀態）
  • 頻譜類：Higher-Order Statistics、Bispectrum（看相位耦合）
6.1

引言 — 大腦是動態系統

許多生理現象都可以從『動態系統』的角度看：

  • 睡眠四階段：訊號趨勢顯著改變，每階段的 EEG 像不同混沌系統。
  • 癲癇發作：發作前周邊神經元的訊號『由混沌漸轉有序』，是發作預測的物理基礎。
  • 運動意念：意圖移動時，α 節律會 desynchronize；動作開始時 β 節律則 synchronize。

混沌數學早於 1890 年代就由 Kolmogorov、Poincaré 等人開啟。1960 年代 Smale 嘗試分類所有典型動態行為；Lorenz、Rössler 系統則展示了『簡單規則 → 複雜行為』的本質。

💡 混沌 ≠ 隨機
混沌：確定性規則 + 對初始條件極敏感 → 看似隨機，但有結構（吸引子、碎形維度）。
隨機：本質無規則。
6.2~6.4

熵 — 度量不確定性

熵 (Entropy) 是衡量訊號『不可預測性』的標準工具：

H(x) = ∫ p(x) · log (1 / p(x)) dx

p(x) 為訊號的機率密度。熵越大代表系統越不可預測。多通道情況下可用聯合熵；若某事件依賴另一事件，可用條件熵。

⚠️ 熵的限制
熵對雜訊極敏感；純雜訊與混沌都可能有高熵，所以熵單獨無法區分『有規律的混沌』與『純隨機』。

6.3 Kolmogorov Entropy

Kolmogorov（或 metric）熵把相空間切成 n-cube，計算軌跡落入各 cube 的聯合機率：

K_n = − Σ P(i_0,...,i_n) ln P(i_0,...,i_n)
K = lim 1/(NT) · Σ (K_{n+1} − K_n)

它估計『從 nT 時刻已知的軌跡 → 預測 (n+1)T 時刻所需的資訊量』。缺點：高維 joint pdf 估計很昂貴，且需大量資料。

6.4 多尺度離散熵 (MFDE)

SampEn、FuzEn、PerEn 都在『單一時間尺度』量化複雜度，但生理訊號天生具多尺度結構。MFDE 透過：

1
Coarse-graining把訊號切成不重疊段，取均值得到不同尺度的時間序列。
2
NCDF 映射透過常態累積分佈函數把 x_j 映到 [0, 1]。
3
線性量化映到 C 個離散等級。
4
建立 fluctuation pattern定義 π_{v_0...v_{m-1}}，計算發生機率。
5
計算 FDispEn對所有 pattern 求 − Σ p ln p。

MFDE 已廣泛用於阿茲海默症、思覺失調、癲癇等的 EEG 複雜度量化。

6.5~6.7

Lyapunov 指數 — 軌跡發散的速率

Lyapunov 指數 λ 是混沌的「數值身分證」：

λ(x_0) = lim 1/n · Σ ln |f'(x_{k-1})|
  • λ > 0 → 系統混沌（誤差以 e^(λn) 放大）
  • λ < 0 → 系統有序、軌跡收斂
  • λ ≈ 0 → 邊緣穩定

紅色與藍色軌跡初值差 ε = 10⁻⁵；觀察是否快速分離。

Quadratic iterator 範例

x(n+1) = α x(n)(1 − x(n))，α = 3.8 時：前 20 點看似隨機；後續呈半規律振盪。這正是 logistic map 在 chaos 區的典型表現。

Kaplan–Yorke 猜想

D_L = m + (1 / |λ_{m+1}|) · Σ_{k=1}^{m} λ_k

其中 m 是最大整數使 λ_1 + ... + λ_m ≥ 0。意涵：由相對容易計算的 Lyapunov 指數，就能反推奇異吸引子的維度。

從時間序列估 Lyapunov — Wolf 演算法

1
建立參考軌跡z_0(0) = x(k_0 τ)，選一個與 z(0) 距離小於 δ 的點。
2
計算對數放大率l_0 = (1/τ) log |z_0(τ) − z_0(0)| / |x(τ) − x(0)|。
3
重新選最近鄰當誤差超過 δ_2 就更新軌跡；確保 δ_1 < 誤差 < δ_2，避免雜訊主導或非線性誤差。
4
平均λ ≈ (1/m) Σ l_j。
6.6~6.7

吸引子維度 與 Detrended Fluctuation Analysis

相空間吸引子

由一維時間序列 x(t) 建立相空間嵌入：

v(t) = (x(t), x(t+τ), x(t+2τ), ..., x(t+(m-1)τ))

關鍵參數：

  • 時間延遲 τ：自相關第一次落到極小，或互訊息第一個極小。
  • 嵌入維度 m：需滿足 m ≥ 2 D_f + 1（Takens 定理）— 確保吸引子在歐式空間不自相交。

🎬 互動：相空間嵌入示意（正弦 vs. 混沌）

把訊號用延遲座標投到 2D 空間：正弦會繪出橢圓；混沌會繪出『折疊』的奇異吸引子；雜訊則是雲狀。

三種維度的關係

維度定義備註
Fractal dim. D_fD_f = lim log M(ε) / log(1/ε)Kolmogorov capacity；單點 D_f=0、線=1、面=2，可為非整數

Detrended Fluctuation Analysis (DFA)

DFA 專門處理非平穩訊號：

1
積分計算 y(k) = Σ (x(i) − x̄)，把訊號變成『隨機漫步』。
2
切窗 + 去趨勢把 y(k) 切成長度 n 的窗，每窗擬合一條多項式 (通常線性) 並減去。
3
計算 F(n)F(n) = √( (1/N) Σ (y(k) − y_fit(k))² )。
4
估冪律指數 αF(n) ∝ n^α；α ≈ 0.5 表白雜訊、α ≈ 1 表 1/f 噪聲、α ≈ 1.5 表布朗運動。
6.7+

Recurrence Quantification Analysis (RQA)

Recurrence plot 把相空間軌跡兩兩比較，距離小於閾值就標一個點。RQA 進一步從這張圖量化系統的規律性與混沌度：

  • Recurrence Rate (RR)：點密度，反映 recurrence 的多寡。
  • Determinism (DET)：對角線段的比例 → 越高代表越接近週期性。
  • Lmax / Lmean：最長／平均對角線段長 → 與 Lyapunov 指數倒數相關。
  • Entropy (ENT)：對角線長度分佈的熵 → 複雜度指標。

RQA 在 EEG 上常用於發作偵測、睡眠分期、認知任務分類。

6.8

高階統計與 Bispectrum

傳統功率譜只到二階，對相位資訊不敏感、對非線性盲。Higher-Order Statistics (HOS) 補上這個缺口：

Bispectrum = 三階累積量的二維傅立葉。可看出『二次相位耦合 (QPC)』 — 即頻率 f_1, f_2 是否與 f_1+f_2 有相位鎖定。
  • 對高斯訊號：bispectrum = 0（高斯只有二階非零）。
  • 對非高斯／非線性訊號：bispectrum 非零，揭露相位耦合。
  • EEG 應用：麻醉深度監測（Bispectral Index, BIS）、癲癇放電辨識、睡眠分期。
6.8+

Approximate Entropy (ApEn)

ApEn 是 Pincus 1991 年提出的『實務化』熵：

1
建立 m-vectorx(i) = (x(i), x(i+1), ..., x(i+m-1))。
2
計算配對對每個 i 算有多少 j 使距離 d[x(i), x(j)] ≤ β。
3
取對數平均ψ_β^m = (1/(N-m+1)) Σ ln ξ_β^m(i)。
4
對 m+1 重複，計算 ApEn = ψ_β^m − ψ_β^{m+1}常取 β = 0.1 ~ 0.25 × σ。

優點：對短資料可用、對 outliers (尖峰) 不敏感、實時應用友善。

用『預測階數』看複雜度

另一個直觀指標：高度相關訊號 (如正弦) 的 AR 模型階數低；雜訊或複雜訊號的 AR 階數高。AIC（Akaike Information Criterion）是常用準則。書中 Figure 6.4 即展示了訊號中段為正弦、首尾為雜訊時，預測階數的變化情形。

6.10

本章摘要

  • 大腦是高度非線性的動態系統；混沌分析提供工具從訊號反推系統指紋。
  • 熵 (Entropy)：通用不確定性度量，但對雜訊敏感，無法區分混沌與雜訊。
  • Kolmogorov 熵需要高維 joint pdf，運算量大；MFDE 是現代多尺度替代品。
  • Lyapunov 指數：軌跡發散的平均對數速率。λ > 0 ↔ 混沌。
  • 由 Kaplan–Yorke 可由 λ 估出吸引子的 Lyapunov 維度。
  • 相空間嵌入需選對 τ（自相關／MI 極小）與 m（≥ 2 D_f + 1）。
  • Correlation/Fractal/Lyapunov 維度三者在合理條件下會吻合。
  • DFA 處理非平穩、估冪律 scaling exponent α。
  • RQA 透過相空間 recurrence 圖量化規律性；Bispectrum 看二次相位耦合。
  • ApEn 對短資料、有 outliers 的場景特別實用。
